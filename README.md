# PrimeTrade Bitcoin Sentiment Analysis

## Objective
This project analyzes the relationship between Bitcoin market sentiment (Fear & Greed Index) and Hyperliquid trader performance to identify patterns that can support better trading decisions.

## Dataset
- Hyperliquid Historical Trader Data
- Bitcoin Fear & Greed Index

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## Analysis Performed
- Data Cleaning
- Data Preprocessing
- Dataset Merging
- Exploratory Data Analysis (EDA)
- Data Visualization
- Key Insights
- Conclusion
